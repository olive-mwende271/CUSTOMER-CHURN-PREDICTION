A complete machine learning project that predicts which customers are likely to churn,
covering data cleaning, integration, exploratory analysis, model training, evaluation, and deployment. 
This helps businesses proactively retain customers and optimize their strategies.

**Features**
Data Cleaning & Integration: Handling missing values, merging datasets, encoding categorical features, and feature scaling.

Exploratory Data Analysis (EDA): Visualizations and insights on customer behavior, churn patterns, and key features.

Model Training: Implemented multiple models including Logistic Regression, Random Forest, and Gradient Boosting.

Model Evaluation: Confusion matrix, accuracy, precision, recall, F1-score, ROC-AUC for model performance comparison.

Deployment: Ready-to-use model deployed for predicting customer churn in real scenarios.
